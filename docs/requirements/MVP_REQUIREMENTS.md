# MVP Requirements — Project Atlas

## Product goal
Provide a high-level executive overview with clickable charts and cards that lead to dedicated workspaces containing full data, trends, source material, and actions.

## MVP workspaces
- Executive Overview
- Intelligence Center
- News Desk
- Human Firewall™ Intelligence
- Standards Watch
- Regulatory Watch
- Progress & Milestones
- Atlas Backlog
- Document Library
- Board Queue
- Design History File / Decisions
- Skills Library

## Executive Overview acceptance criteria
- Shows high-level indicators only.
- Each visualization is clickable.
- Each click opens the correct dedicated workspace.
- The overview does not duplicate all detail-page content.
- Metrics are sourced from structured data or clearly labeled as demo data.
- Layout remains usable on desktop and tablet widths.

## Intelligence acceptance criteria
- Shows both an executive summary and the underlying news/articles.
- Every article can retain an original-source link.
- Articles can be categorized and filtered.
- Disposition supports Archive, Monitor, Board Review, and Integration Candidate.
- Human judgment remains required for methodology changes.

## Quality acceptance criteria
- Brand-compliant header and colors.
- No screenshot-based production assets.
- Correct website: `solarisadvisoryai.com`.
- Keyboard-accessible navigation.
- No broken links or console errors.
- README contains local run instructions.
