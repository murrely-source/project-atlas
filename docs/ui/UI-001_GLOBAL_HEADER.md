# UI-001 — Global Application Header

## Source of truth
Solaris Brand Guide — Dark Edition.

## Required elements
- Approved clean Solaris symbol asset.
- SOLARIS wordmark.
- `AI RISK & GOVERNANCE ADVISORY` subline.
- Vertical divider.
- Dynamic workspace title.
- Correct website in the application metadata area where applicable.

## Prohibited
- Cropped Brand Guide screenshots.
- Invented logos or substitute marks.
- Unapproved colors or fonts.
- Embedded page titles inside a logo image.

## Acceptance criteria
- Header remains consistent across all workspaces.
- Brand area is reusable.
- Workspace title changes independently.
- Header is responsive.
