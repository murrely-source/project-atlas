# Solaris Lucerna production assets

The approved Solaris Lucerna cinematic sunrise-and-horizon artwork is visible inside the brand-standard reference boards, but no standalone production asset is currently present in the repository.

Add the final, approved, web-optimized hero artwork at:

`app/assets/brand/solaris-lucerna-hero-approved.webp`

Requirements:

- Export from the approved master artwork, not a crop of a brand-guide board.
- Preserve the approved composition, color, and proportions.
- Provide sufficient dimensions for responsive desktop and high-density displays.
- Do not substitute stock imagery, generated imagery, or a recreated sunrise.

The homepage intentionally shows a labeled placement panel until this file is approved and supplied.
